# 项目简介

基于Spring Boot + Uniapp的奶茶小程序（仿蜜雪+论文）是一个旨在为奶茶店提供高效管理服务的微信小程序后端系统。该系统结合了Spring Boot的强大后端功能和Uniapp的前端跨平台开发能力，为奶茶店提供全面、便捷的管理体验。

## 核心功能

- **菜品管理**：支持对奶茶店菜品进行分类、编辑、添加和删除等操作。
- **订单管理**：实现对订单的生成、处理、查询和跟踪等功能。
- **用户管理**：管理用户信息，包括会员信息的添加、修改和查询。
- **微信支付**：集成微信支付接口，实现订单的在线支付功能。
- **收银管理**：支持收银员操作，快速处理订单，实现现金与移动支付的对接。
- **会员管理**：会员积分管理，积分兑换功能，提升用户粘性。
- **客户管理**：记录客户消费记录，提供客户数据分析。
- **财务管理**：实现收入、支出、成本等财务数据的统计和分析。
- **系统设置**：管理后台的权限设置，日志管理等。

## 适用场景

- 奶茶店
- 小型餐饮企业
- 跨境电商

## 技术特点

- **Spring Boot**：后端采用Spring Boot框架，保证了系统的稳定性和高效性。
- **Uniapp**：前端采用Uniapp框架，实现了一次编写，多端运行的跨平台能力。
- **微信支付**：集成微信支付，方便用户支付，提高支付效率。
